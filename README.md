# Abhishek-portfolio

A clean and interactive personal portfolio website designed to showcase my projects, skills, and experience.

##  Main Features

- **Interactive 3D Elements:** Includes a 3D Skills Cube and a rotating Profile Coin using Three.js and CSS transforms.
- **Modern Design:** Features a dark-mode aesthetic with smooth scroll animations and glassmorphism effects.
- **Fully Responsive:** Works seamlessly on desktops, tablets, and mobile phones with a functional hamburger menu.
- **Dynamic Background:** A responsive cosmic background powered by Three.js.

##  Built With

- **HTML5:** Page structure and semantic content.
- **CSS3:** Custom styling, layout (Flexbox/Grid), and animations.
- **JavaScript:** Interactive components, scroll logic, and mobile navigation.
- **Three.js:** Used for the 3D background effects.

##  How to Run the Project

1. Download or clone the repository.
2. Open the project folder.
3. Since the project uses Three.js, it is best viewed through a local server:
   - If using VS Code, right-click `index.html` and select **"Open with Live Server"**.
   - Alternatively, use `npx http-server` or any other local server tool.
4. Open your browser and navigate to the local address (usually `http://127.0.0.1:5500`).
